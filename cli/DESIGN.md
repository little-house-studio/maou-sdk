# Maou CLI

- 配色：
  - 主色：黑色
- UI风格：
  - 沉浸式
  - 必要ascii图标
  - 用背景色分块
  - 信息密集，分割线距离小

- 输入框（组件，最底部固定第二位置）

## 上下文窗口（对话区）
- 整个界面95%面积应该沉浸式，是最重要的区域
- 可以滚轮滑动窗口
- 角色内容（背景块显示颜色，例如user、assistant、system）
  - 卡片渲染
    - 代码气泡，全彩显示
    - md渲染，
    - 工具卡片折叠与展开
- 显示的内容：
  - content内容：完整显示
    - md格式、代码结构渲染
  - 工具调用部分：
    - 工具卡片：每个工具都是一个收纳卡片
      - 创建文本、编辑：显示附近10行，收纳
      - 阅读工具、仅仅显示工具调用记录，默认收纳


## 输入框
- 最底部往上数固定第二位置，状态栏上面
- 默认一行，自动根据文本内容自适应改高度（自适应）
- 按键
  - alt+enter换行
  - enter发送指令
  - alt+退格，快速文字退格
  - shift+点击文字选中区域文字
  - 粘贴上去过长的内容会被占位符替换显示（实际上发送出去是完整内容非占位符），删除占位符是这一整块占位符删除掉。


## 状态栏
- 自动布局
- 从左往右
  - 当前agent状态
  - 后台终端数
  - 上下文占用量
    -  [上下文占用token/模型总token][变色条形显示][占用率][最近20轮平均缓存率]


## 功能菜单弹窗
- ctrl+k 打开功能菜单弹窗，esc/ctrl+k取消/返回，回车打开选项子菜单
