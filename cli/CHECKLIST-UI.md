## A. 本轮已改（1–6）

| # | 项 | 行为 | 建议自测 |
|---|----|------|----------|
| 1 | **滚轮一格一行** | `scrollChat` step 固定为 **1** | 对话区滚轮：应每次只动一行，不再跳很多行 |
| 2 | **加载会话 · 工具卡** | `session-loader` 恢复 `toolCalls` + `role:tool` 结果；卡片 **默认收纳** | `/sessions` 或启动恢复历史：应看到黄底工具名 + ▶，点开才有结果 |
| 3 | **超 10 行自动折叠** | user 灰块 / assistant 正文 / system / 工具结果超约 10 **显示行**折叠，点击展开/收起 | 发长文或加载长回复：见 `▼ 展开全文`，点后 `▲ 收起` |
| 4 | **上下文拖选（网页感）** | 对话区按下拖动蓝底选区；近上下缘 **自动滚 1 行**；拖选中滚轮滚动不丢选；松手 OSC52 复制 | 在回复上拖选一段；拖到顶/底应慢慢滚；松手可粘贴 |
| 5 | **更多网页交互** | 见下表 B | 逐项点 |
| 6 | **排版 / 显示** | logo 列对齐；user 整块灰矩形；loop 分隔单行；`&quot;` 解码；工具默认收纳 | 看 user 灰块、◈/▸/▣ 是否一列、loop 是否一行 |

## E. 建议验收顺序

1. `cd maou-sdk/cli && npm run dev`  
2. 滚轮：是否 **一行一行**  
3. 发/载长消息：是否 **10 行折叠**、点击展开  
4. 加载旧会话：是否有 **工具卡 ▶**、点开有结果  
5. 对话区 **拖选 + 顶底自动滚 + 复制**  
6. user **整块灰矩形**、logo 列、loop 单行  
7. 流式时是否仍能打字/滚动（大致可用即可）
