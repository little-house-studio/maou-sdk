# 通用工具 (`core/common`)

---

## 1. ConfigStore —— 配置管理

多层级合并（项目 > 用户 > 默认值），支持热重载。

### 导入

```ts
import { loadConfig } from 'maou-agent/core/common'
```

### 加载配置

```ts
const config = loadConfig({
  projectConfigPath: 'project_config.json',      // 项目级配置
  userConfigPath: '~/.maou/user_config.json',    // 用户级配置
  presetDir: 'core/llm/presets',                 // preset 扫描目录
})
```

### 配置层级（deep merge）

```
项目配置 (project_config.json)
  ↓ 覆盖
用户配置 (~/.maou/user_config.json)
  ↓ 覆盖
代码默认值
```

### AppConfig 结构

```ts
interface AppConfig {
  api: {
    presets: PresetConfig[]
    defaultPreset: number          // 默认预设索引
    agentRoundLimit: number        // 默认 50
    promptRoot: string             // 默认 'ROLE/default'
    promptEntrypoint: string       // 默认 'SYSTEM.md'
    userEntrypoint: string         // 默认 'USER.md'
    toolSchemasFile: string        // 默认 'TOOL.jsonc'
    contextSettings: {
      thresholdPercent: number     // 触发压缩阈值，默认 70
      keepRecentPercent: number    // 保留近期比例，默认 25
    }
    pluginSettings?: PluginSettings
  }
  security: {
    sandboxMode: string            // 默认 'normal'
  }
  hub: {
    enabled: boolean
    port: number
  }
  server: {
    port: number
    host: string
  }
}
```

## 2. ProjectManager —— 项目管理

### 导入

```ts
import { ProjectManager } from 'maou-agent/core/common'
```

### 构造函数

```ts
const manager = new ProjectManager({ maouRoot: '~/.maou' })
```

### 方法

| 方法 | 签名 | 说明 |
|------|------|------|
| `list` | `() => ProjectInfo[]` | 列出已注册项目 |
| `add` | `(projectPath: string) => void` | 注册新项目 |
| `remove` | `(projectName: string) => boolean` | 移除项目 |
| `getCurrent` | `() => ProjectInfo \| null` | 获取当前活跃项目 |

### 使用示例

```ts
const manager = new ProjectManager({ maouRoot: '~/.maou' })

// 注册项目
manager.add('/path/to/my-project')

// 列出
const projects = manager.list()
// [{ name: 'my-project', path: '/path/to/my-project', ... }]

// 当前项目
const current = manager.getCurrent()

// 移除
manager.remove('old-project')
```
