# 上下文层 (`core/context`)

---

## SessionStore —— 会话持久化

基于 JSONL 的会话持久化系统，每个 session 一个文件。

### 导入

```ts
import { SessionStore } from 'maou-agent/core/context'
// 或从 agent 层重新导出
import { SessionStore } from 'maou-agent/core/agent'
```

### 构造函数

```ts
const store = new SessionStore(sessionsDir: string)
```

### 方法表

| 方法 | 签名 | 说明 |
|------|------|------|
| `createSession` | `(sessionId: string, overrideMeta?: Partial<SessionMeta>) => SessionData` | 创建新会话 |
| `loadSession` | `(sessionId: string) => SessionData` | 加载会话全部数据 |
| `listSessions` | `() => SessionListItem[]` | 列出所有会话摘要 |
| `appendMessage` | `(sessionId: string, msg: object) => void` | 追加一条消息 |
| `appendTrace` | `(sessionId: string, trace: object) => void` | 追加一条追踪记录 |
| `clearSession` | `(sessionId: string) => void` | 清空会话内容 |
| `deleteSession` | `(sessionId: string) => void` | 删除会话文件 |

### 使用示例

```ts
const store = new SessionStore('~/.maou/sessions')

// 创建
store.createSession('abc123', { agent_name: 'maou', title: '新的对话' })

// 读取
const all: SessionListItem[] = store.listSessions()
const session: SessionData = store.loadSession('abc123')

// 消息操作
store.appendMessage('abc123', { role: 'user', content: 'hello' })
store.appendTrace('abc123', { type: 'tool_call', name: 'read_file', params: {} })

// 生命周期
store.clearSession('abc123')
store.deleteSession('abc123')
```

### 核心类型

```ts
interface SessionData {
  id: string
  title: string
  agentName: string
  messages: SessionMessage[]
  trace: SessionTrace[]
  createdAt: string
  updatedAt: string
  lastPrompt: string
  lastRawResponse: string
  raw_data: { rounds: unknown[] }
}

interface SessionListItem {
  id: string
  title: string
  updatedAt?: string
  messageCount: number
  lastMsgAt: string
}

interface SessionMessage {
  role: string
  content: string
  created_at: string
  [key: string]: unknown
}

interface SessionMeta {
  id: string
  title: string
  agent_name: string
  created_at: string
  updated_at: string
  last_prompt: string
  last_raw_response: string
}
```
