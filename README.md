# maou-sdk

Maou Agent 官方 SDK — 可复用组件集合

## 包结构

| 包名 | 说明 | 状态 |
|------|------|------|
| `@little-house-studio/llm` | LLM 通信层 | ✅ 已完成 |
| `@little-house-studio/context` | 上下文管理 | 🚧 开发中 |
| `@little-house-studio/tools` | 工具系统 | 🚧 开发中 |
| `@little-house-studio/core` | 核心类型 | 🚧 开发中 |
| `@little-house-studio/agent-harness` | Agent 运行时 | 📋 计划中 |
| `@little-house-studio/cli` | CLI 工具 | 📋 计划中 |
| `@little-house-studio/webui` | Web UI | 📋 计划中 |

## 安装

```bash
npm install @little-house-studio/llm
```

## 开发

```bash
pnpm install
pnpm build
```
