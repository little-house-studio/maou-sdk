## 使用指引

- 项目管理工具。仅在用户明确要求管理项目时使用。
- action=list：查看所有项目；action=create：创建项目；action=disband：解散项目。
- action=add-agent：为项目创建新 Agent，需指定 agent_name、role、preset、permission 等。
- preset 预设：default、frontend、backend、tester，快速设定 Agent 职能。
- permission 权限：full（完全控制）、restricted（受限）、observer（只读观察）。
- 不要主动创建项目——只在用户要求时操作。
