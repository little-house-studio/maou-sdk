## 使用指引

- 团队管理工具。仅在用户要求管理 Agent 团队时使用。
- action=list：查看团队成员；action=create：创建新 Agent。
- action=stop：停止 Agent；action=interrupt：中断 Agent 当前执行。
- action=message：向 Agent 发消息；action=insert/remove：添加/移除团队成员。
- 新建 Agent 时需指定 name、role，可选 description、personality、team 等。
- scope 控制作用域：system（系统级）、project（项目级，默认）。
