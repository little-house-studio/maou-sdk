## 使用指引

- old_text 必须是文件中实际存在的精确文本（含空格、缩进、换行）。
- **默认**：old_text 必须在文件中**唯一**匹配，只替换该处。
- **批量替换**：传 `"replace_all": true` 可替换全部相同文本（变量重命名等）。
- 如果 old_text 匹配不到，先用阅读工具查看文件实际内容，再复制精确文本。
- 不要用 edit_file 做大规模重写——多次小替换不如一次 write_file 全量替换。
- 替换后建议用阅读工具确认修改结果是否符合预期。
- 同一文件需要多处不同修改时，按从文件末尾到开头的顺序操作，避免行号偏移。
- 修改时保持缩进风格与原文件一致（空格/Tab、缩进宽度）。
