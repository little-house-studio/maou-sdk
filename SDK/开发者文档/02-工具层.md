# 工具层 (`core/tools`)

工具系统提供工具的注册、发现与执行管道。所有工具必须继承 `Tool` 基类。

---

## 1. Tool 抽象基类

### 导入

```ts
import { Tool, createToolResponse } from 'maou-agent/core/tools'
```

### 定义新工具

```ts
class ReadFileTool extends Tool {
  readonly definition: ToolDefinition = {
    name: 'read_file',
    aliases: ['read', 'cat'],
    description: '读取文件内容',
    parameters: {
      type: 'object',
      properties: {
        file_path: { type: 'string', description: '文件路径' },
        start_line: { type: 'integer', description: '起始行' },
      },
      required: ['file_path'],
    },
    allowedModes: null,  // null = 所有模式可用
  }

  async execute(
    params: Record<string, unknown>,
    ctx: ToolContext,
  ): Promise<ToolResponse> {
    return createToolResponse(true, `已读取 ${params.file_path}`)
  }
}
```

### 核心类型

```ts
interface ToolDefinition {
  name: string
  aliases: string[]
  description: string
  parameters: JsonSchema          // JSON Schema 格式
  allowedModes: string[] | null   // null = 全部可用
}

interface ToolContext {
  sessionId: string
  projectRoot: string
  promptRoot: string
  sandboxRoot: string
  sandboxMode: string
  agentName: string
  agentMode: string
  pluginSettings: Record<string, unknown>
  workingDir: string
}

interface ToolResponse {
  ok: boolean
  message: string
  displayEvents: Record<string, unknown>[]
  payload: Record<string, unknown>
  background: boolean    // true = 后台执行，不阻塞
  images: { mimeType: string, data: string }[]
}
```

### 辅助函数

```ts
// 创建标准 ToolResponse，只需指定 ok + message，其余有默认值
function createToolResponse(
  ok: boolean,
  message: string,
  extras?: Partial<ToolResponse>,
): ToolResponse
```

## 2. ToolRegistry —— 工具注册表

```ts
import { ToolRegistry, registerBuiltins } from 'maou-agent/core/tools'
```

### 方法表

| 方法 | 签名 | 说明 |
|------|------|------|
| `register` | `(tool: Tool) => void` | 注册工具（同时注册别名） |
| `get` | `(name: string) => Tool \| undefined` | 按名称/别名查找 |
| `listDefinitions` | `() => ToolDefinition[]` | 列出所有定义 |
| `getAll` | `() => Map<string, Tool>` | 获取全部工具 |
| `setSchemasFile` | `(path: string \| null) => void` | 设置 TOOL.jsonc 路径 |
| `loadSchemasFrom` | `(filePath: string) => void` | 从 JSONC 加载权限/模式配置 |

### 使用示例

```ts
const registry = new ToolRegistry()

// 注册内置工具
registerBuiltins(registry)

// 注册自定义工具
registry.register(new MyCustomTool())

// 查找
const tool = registry.get('read_file')

// 列出所有定义
const defs: ToolDefinition[] = registry.listDefinitions()
```

## 3. ToolExecutor —— 工具执行管道

封装工具调用、超时处理与并发控制。

```ts
import { ToolExecutor } from 'maou-agent/core/tools'

const executor = new ToolExecutor(registry, {
  defaultTimeoutMs: 30_000,  // 默认 30s 超时
  maxConcurrency: 10,        // 最大并发
})

// 执行单个工具
const result = await executor.executeSingle(
  { id: 'call_1', name: 'read_file', parameters: { file_path: '/a.txt' } },
  ctx,
)

// 批量并行执行
const results = await executor.executeBatch(
  [{ id: 'call_1', name: 'read_file', parameters: { file_path: '/a.txt' } }],
  ctx,
)
```

### ToolExecutionResult

```ts
interface ToolExecutionResult {
  toolCall: ToolCall         // 原始调用
  events: Record<string, unknown>[]
  result: ToolResponse
}

interface ToolCall {
  id: string
  name: string
  parameters: Record<string, unknown>
}
```
